## Freenove Robot Dog Fork

> A Robot Dog Kit for Raspberry Pi.

### Purpose

The purpose of this fork is to create an autonomous version of the Dog, whith a minimal `Server` mode, but a bigger
self-working mode. No AI included :)

### Download original

The original kit was provided by [Freenove](https://www.freenove.com/).

Run following command to download all the files in this repository.

	`git clone https://github.com/Freenove/Freenove_Robot_Dog_Kit_for_Raspberry_Pi.git`
